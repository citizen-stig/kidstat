KidStat

Allows you to explore World Health Organization's data about child growth standards without diving into data files!

How it works?

World Health Organization provides standards for child growth, that have been collected over time. It's possible to categorize measurement by groups

When you enter observation value, applications looks over available standards and categorize it using a following table:

Percentile Range Category Name
[0 - 5]Very Low
[6 - 15]Low
[16 - 25]Below Average
[26 - 75]Average
[76 - 85]Above Average
[86 - 95]High
[96 - 100]Very High

Technologies used

MongoDB is used for data storage. Flask, a Python microframework handles logic and provides REST API and authorization. React.js is used for client side logic. Vanilla Bootstrap is used for styling.
